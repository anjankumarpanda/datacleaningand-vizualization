# Data Cleaning & Visualization Project

## Objective
Clean raw data and generate useful insights.

## Steps Performed
- Handled missing values
- Removed duplicates
- Treated outliers
- Converted data types
- Visualized data using charts

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Output
- Clean dataset
- Graphs and charts
- Insights from data

## How to Run
1. Install libraries:
   pip install -r requirements.txt

2. Run program:
   python main.py
